# Homebrew Tap for WorkTools

Official Homebrew tap for [WorkTools](https://github.com/khaledalam/worktools).

## Installation

```bash
brew install khaledalam/worktools/worktools
```

Or:

```bash
brew tap khaledalam/worktools
brew install --cask worktools
```
